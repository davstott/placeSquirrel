place squirrel

Please do not actually use this service for anything yet, it's not in any way optimised and would hammer my server into the ground.

An easy to use service that provides sufficient pictures of squirrels for your websites. It's intended to be used to provide placeholders to illustrate design and layout before real image assets become available, although it helps if you like squirrels too.

To use it: <img>http://davstott.me.uk/squirrel/300/400</img>
or if you want image 'f': http://davstott.me.uk/squirrel/f/300/400

Written by Dav Stott, images provided by the internet's photographers (image attribution). The idea is to be like placekitten, but with squirrels. Fork me at github.
